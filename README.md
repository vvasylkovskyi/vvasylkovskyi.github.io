# Viktor Vasylkovskyi - Portfolio

A modern, responsive portfolio website built with React, TypeScript, and Tailwind CSS.

## Development

Install dependencies:
\`\`\`bash
npm install
\`\`\`

Run development server:
\`\`\`bash
npm run dev
\`\`\`

## Build for GitHub Pages

Build the project:
\`\`\`bash
npm run build
\`\`\`

This will create a `dist` folder with the compiled `index.html` and assets.

## Deploy to GitHub Pages

1. Push your code to GitHub
2. Copy contents of `dist` folder to your GitHub Pages repository
3. Commit and push
4. Your portfolio will be live at `https://yourusername.github.io`

Alternatively, you can configure GitHub Actions to automatically build and deploy from the `dist` folder.

## Customize

- Update your photo: Replace `public/professional-headshot.png`
- Edit content: Modify `src/App.tsx`
- Change colors: Update `tailwind.config.js`
- Add experiences: Edit the work experience section in `src/App.tsx`
